# Omnitext

> A lightweight, scalable real-time messaging library for Android apps.

Omnitext powers 1:1 chats, group conversations, push notifications, and end-to-end encryption,
with a strong focus on performance and user privacy.

---

## ✨ Features

- 💬 **1:1 and group chats** — real-time messaging between users or within groups
- 🔔 **Push notifications** — instant delivery even when the app is in the background
- 🔐 **End-to-end encryption** — messages are private by design
- ⚡ **Lightweight & scalable** — minimal footprint, built to grow with your app
- 🛡️ **Privacy-first** — no unnecessary data collection or third-party tracking

---

## 🛠️ Tech Stack

- **Language:** Kotlin
- **Platform:** Android
- **Build system:** Gradle (Kotlin DSL)

---

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest stable)
- Android SDK 21+
- JDK 11+

### Clone the repository

```bash
git clone https://github.com/mattiagabellieri/Kakoste.git
cd Kakoste
```

### Build

Open the project in Android Studio and sync Gradle, or run:

```bash
./gradlew assembleDebug
```
